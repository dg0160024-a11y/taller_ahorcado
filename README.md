# 🎮 JUEGO DEL AHORCADO EN JAVA 🖥️ 🔥

Este proyecto tiene como objetivo y finalidad desarrollar e implementar el clásico juego del Ahorcado utilizando el lenguaje de programación Java, aplicando conceptos fundamentales de lógica de programación y programación orientada a objetos.

A través de este trabajo, se busca comprender cómo interactuar con el usuario mediante consola, manejar palabras aleatorias, validar entradas y controlar el flujo de un juego utilizando estructuras de control y estructuras de datos.

También se implementan mecánicas importantes como el control de vidas, validación de letras repetidas, actualización dinámica de palabras y representación gráfica básica mediante ASCII Art.

En resumen, este proyecto demuestra cómo desarrollar una aplicación interactiva en Java, fortaleciendo habilidades en lógica, validación de datos y organización del código para la creación de programas más completos y dinámicos.

## ⚙️ ¿Qué se aprende en este proyecto?
- Uso de variables y estructuras de control
- Manejo de Arrays y ArrayList en Java
- Uso de Scanner para entrada de datos
- Generación aleatoria con Random
- Validación de letras ingresadas
- Manipulación de cadenas y caracteres
- Uso de métodos para organizar el código
- Implementación de lógica de videojuegos básicos
- Representación gráfica en consola con ASCII Art
- Desarrollo de programas interactivos en Java
